Katia Marangoni — Site Link na Bio
Site de página única desenvolvido para ser usado como link na bio do Instagram de Katia Marangoni, Esteticista Corporal.
O que é
Landing page com apresentação profissional, lista de tratamentos com benefícios e botões de agendamento direto pelo WhatsApp.
Tecnologias

HTML5 e CSS3 puro — sem frameworks, sem dependências
Fonte: Cormorant Garamond + Inter (Google Fonts)
Imagem embutida em base64 — sem necessidade de CDN externo

Estrutura
katia-site/
├── index.html
└── vercel.json
Como publicar no Vercel

Faça o clone do repositório
Acesse vercel.com e crie uma conta
Clique em Add New Project → Import Git Repository
Selecione este repositório
Clique em Deploy — sem nenhuma configuração adicional

A URL gerada (seu-projeto.vercel.app) pode ser usada diretamente na bio do Instagram.
